Velock

Discover

Velock is a web application designed to simplify looking up availability data for Santander Bikes. Leveraging the TfL Open API, Velock provides up-to-the-minute availability data for bikes and docking stations across the city. Users can create personalised lists of frequently used stations for quick access. Additionally, Velock offers powerful search functionality, allowing users to locate specific stations or discover nearby docking options based on an address.
